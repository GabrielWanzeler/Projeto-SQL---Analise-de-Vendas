# Projeto-SQL---Analise-de-Vendas

## Objetivo

Analisar dados de vendas utilizando SQL para responder perguntas de negócio
e praticar consultas com agregações, filtros e subqueries.

## Dataset

Base de vendas fictícia (Superstore) obtida no Kaggle.

## Ferramentas

* SQL
* PostgreSQL
* pgAdmin
* GitHub

## Estrutura do Projeto

projeto-sql-vendas/
│
├── dataset/
│ └── superstore.csv
│
├── sql/
│ ├── create_table.sql
│ └── perguntas_negocio.sql
│
├── README.md

Cada arquivo SQL está comentado e pode ser executado diretamente no PostgreSQL.

## Perguntas de Negócio Respondidas

1\. Qual o faturamento total da empresa?

2\. Qual o faturamento por categoria?

3\. Quais são os produtos mais vendidos?

4\. Quais os clientes que mais gastaram?

5\. Qual o Ticket médio por pedido?

6\. Qual o faturamento por estado?

7\. Qual o numero de pedidos por cliente?

8\. Qual a média de itens por pedido?

9\. Qual o mês de maior faturamento?

10\. Quais categorias faturam acima da média das categorias?
